Данный web-ресурс содержит в себе лабораторный практикум по JavaScript. На главной странице расположены ссылки на лабораторные работы, которые оформлены в виде карточек с написаной в центре основной темой, изученой в каждой лабораторной работе. Порядок работ идет построчно.

Всего в практикуме 5 лабораторных работ:

  1. Изучение объекта Window
  2. Изучение объекта Document
  3. Изучение объекта Form
  4. Изучение объекта Image
  5. Изучение объекта Style

При открытии любой из лабораторных работ вы увидете страницу, верху которой будет находиться навигационое меню. Номер открытой в данный момент работы обозначен другим цветом. Кнопка с надписью "Содержание" открывает содержание данной страницы с ссылками на соответствующие главы в лабораторной работе. При нажании на логотип "JavaScript" вы вернетесь на главную страницу.

После окончания теории в каждой лабораторной работе идет раздел "Практика", который является обязательным к выполнению. Быстрый переход к практике осуществляется с помощью содержания.

В так называемом "подвале" веб-страницы по центру расположена ссылка на сайт для онлайн-доступа к лабораторному практикуму.
